This code defines functions for calculating the digital root and digital sum of a number in a given base. The `digSum` function takes a base and a number and returns the sum of its digits in the specified base. The `digRoot` function takes a base and a number and returns a pair consisting of the digital root and the original number. The `main` function demonstrates the usage of these functions by calculating the digital roots of several numbers in base 10 and printing the results.
