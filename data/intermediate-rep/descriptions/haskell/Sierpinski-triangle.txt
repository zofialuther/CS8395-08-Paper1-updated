This code generates a Sierpinski triangle pattern using the Rule 90 cellular automaton. The `sierpinski` function takes an integer as input and produces a string representation of the Sierpinski triangle with the specified number of iterations. The `rule90` function utilizes the XOR operation to generate the next iteration of the pattern, and the `spacing` function handles the formatting of the output. The main function prints the Sierpinski triangle with 4 iterations to the console.
