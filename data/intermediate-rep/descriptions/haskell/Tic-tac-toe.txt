This code is a program to play a game of Tic-Tac-Toe. It includes functions for checking if a player has won, determining the best move for the computer player, allowing the user to place their pieces on the board, and implementing an AI for the computer player. The main program interacts with the user to determine the game type and which player starts, and then plays the game accordingly. The user can choose to play against another human, against the computer, or have the computer play against itself. The program also includes a usage function to provide instructions on how to run the program with different options.
