This Python code is a program that generates a table displaying Chinese zodiac details for a given list of years. It includes functions for creating the zodiac table, transposing the data, and generating the wiki table. The zodiac details are derived from traditional Chinese strings and mapped to the corresponding years based on the Chinese zodiac system. The program outputs the zodiac details for specific years, including the heavenly stems, terrestrial branches, elements, symbolic animals, and yin-yang properties.
